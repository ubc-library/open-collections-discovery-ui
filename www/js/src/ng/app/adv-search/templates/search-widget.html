Advanced Options

Attribute Name Accepts Default Description
data-placeholder String Search the Open Collection...
Text appearing in search box prior to user typing
data-color Colour hexcode #002145
Theme colour for buttons in search widget
data-inline Boolean false Default sends user to Open Collections landing page for search results. 'True' returns search results via AJAX within hosting page, underneath searchbox.
data-scope String None Scoped collections search. Only applicable for inline searches. Current valid values are: chung, berkeley
data-size Integer 5 Number of search results returned per page. Only applicable for inline searches.
data-debug Boolean false Outputs some debug code to console. Only application for inline searches.
